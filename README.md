# BOOL Language

Educational object-oriented programming language featuring a custom compiler and bytecode interpreter written in Java.

## About

BOOL is an educational programming language developed for the Language and Programming Paradigms course.

The project implements a complete execution pipeline composed of:

- A source code compiler
- A custom bytecode format
- A stack-based bytecode interpreter
- Runtime execution structures

The language supports arithmetic operations, variables, conditionals, object-oriented constructs, method invocation, object manipulation, and execution flow control.

Main topics explored in this repository:

- Compiler construction
- Bytecode generation
- Stack-based virtual machines
- Runtime systems
- Programming language design
- Object-oriented language implementation
- Parsing using regular expressions
- Execution contexts and memory structures

The codebase was developed using:

- Java
- Object-oriented programming
- Stack-based execution
- Regular expressions
- Custom runtime structures

## Compiler

The compiler is responsible for translating BOOL source code into an intermediate bytecode representation.

Implemented features:

- Variable declarations
- Arithmetic expressions
- Comparisons
- Conditional statements
- Method calls
- Object creation
- Bytecode generation

---

## Interpreter

The interpreter executes BOOL bytecode using a stack-based virtual machine architecture.

Implemented features:

- Stack execution model
- Execution contexts
- Arithmetic operations
- Comparisons
- Variable storage and loading
- Method invocation
- Object field manipulation
- Control flow execution

---

## Runtime Structures

The runtime system includes custom execution and object management structures.

Main components:

- `ExecutionContext`
- `BoolInterpreter`
- `BoolClasse`
- Bytecode command system
- Object manipulation structures

---

## Example

BOOL source code:

```text
a = 10
b = 20
c = a + b
```

Generated bytecode:

```text
const 10
store a
const 20
store b
load a
load b
add
store c
```
