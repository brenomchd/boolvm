# boolvm

BoolVM is an academic programming language implementation written in Java. It includes a compiler that translates `.bool` source files into `.boolc` bytecode and an interpreter that executes the generated bytecode.

## Features

- Object-oriented class definitions with attributes and methods
- Method calls, assignments, arithmetic operations, and conditional branches
- Prototype-based object delegation through `_prototype`
- A simple bytecode interpreter with basic garbage collection

## Build and Run

From the project root, compile all Java files:

```sh
javac $(find . -name "*.java")
```

Compile the sample program:

```sh
java compiler.BoolCompiler source.bool output.boolc
```

Run the generated bytecode:

```sh
java interpreter.BoolInterpreter output.boolc
```

`BoolCompiler` appends to the output file, so clear `output.boolc` before recompiling if you want a fresh bytecode file.
